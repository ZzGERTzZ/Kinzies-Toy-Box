set_never_turn_on_player(npc_name[, flag])
Parameters:
npc_name (string) name of the NPC
flag (boolean, optional) set to true to have the npc ignore when the player shoots him, false for him to take umbrage
 
Returns:
none
Description:
Enable/disable turn on the player for an NPC.
