on_game_time_trigger(func_name, seconds[, minutes[, hours[, days[, months[, years]]]]])
Parameters:
func_name (string) name of the function, or "" to unregister the event
seconds (integer) number of seconds to delay
minutes (integer, optional) number of minutes to delay (defaults to 0)
hours (integer, optional) number of hours to delay (defaults to 0)
days (integer, optional) number of days to delay (defaults to 0)
months (integer, optional) number of months to delay (defaults to 0)
years (integer, optional) number of years to delay (defaults to 0)
 
Returns:
none
Description:
Register a callback function to be called after a certain amount of game time (not real time) has elapsed.
