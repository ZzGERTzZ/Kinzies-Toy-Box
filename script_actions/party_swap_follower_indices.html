party_swap_follower_indices(index1, index2)
Parameters:
index1 (integer) index of the first follower
index2 (integer) index of the second follower
 
Returns:
none
Description:
Swap the follower indices of two player followers.

This function is only supported in single player. In coop, both players have just one follower, so swapping follower indices makes no sense.
