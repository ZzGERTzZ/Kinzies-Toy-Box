radio_set_sing_along_track()
Parameters:
none
 
Returns:
none
Description:
Sets the sing along track to be used by the player's current

persona for a given track. Can be used to enable a singalong during a mission that can later be cleared so it is not used during regular gameplay.

track_name: The name of the track the player will sing along to. singalong_track_name: The name of the track to use for the sing along.
