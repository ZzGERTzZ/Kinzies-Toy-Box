qte_start_m02_skyqte_01(player_name, function_name[, anim_partner_name[, grapple_nav_name]])
Parameters:
player_name (string) player to activate the QTE on
function_name (string) name of the LUA function to call on completion
anim_partner_name (optional, string) name of the animation partner for the QTE
grapple_nav_name (optional, string) name of a navpoint to slide into position during the grapple
 
Returns:
none
Description:
Starts the loaded QTE on both clients.
