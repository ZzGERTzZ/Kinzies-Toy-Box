on_hud_screen_exit(func_name, screen_name)
Parameters:
func_name (string) name of function or "" to unregister the event
screen_name (string) name of the screen to fire the callback for
 
Returns:
none
Description:
Register a callback function for when a ui screen is exited from

When the callback function is executed, the following parameters are passed to it: - (string) the name of the screen that was exited - (string) the name of the player that exited from the screen
