player_human_shield_lock(player_name[, locked])
Parameters:
player_name (string) name of the player to lock the human shield state of.
locked (bool, optional) true to lock the human shield state, or false to unlock (default true)
 
Returns:
none
Description:
Prevent a player from entering/leaving a human shield state by pressing the 'Y' or right bumper button.
