boss_battle_matt_cheats_start([cheat_index[, cheat_duration[, cheat_period[, delay_cheat_activation]]]])
Parameters:
cheat_index (int, optional) index denoting which cheat to continually do
cheat_duration (int, optional) overrides the number of milliseconds to play the cheat for
cheat_period (int, optional) overrides the number of milliseconds between cheats
delay_cheat_activation (bool, optional) determines if there will be a delay(Cheat_boss_buffer_time) before the cheat starts
 
Returns:
none
Description:
Enables Matt's cheats, optional peramaters will enforce a specific cheat
