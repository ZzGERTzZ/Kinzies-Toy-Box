on_notoriety_event(func_name[, name])
Parameters:
func_name (string) name of the function, or "" to unregister the event
name (string, optional) name of the player (defaults to "#PLAYER1#")
 
Returns:
none
Description:
Register a callback function for when the player's notoriety increases due to a notoriety event.

When the callback is executed, the following parameters are passed to it:
 - (string) name of the notoriety event that occurred
 - (integer) increase in police notoriety
 - (integer) increase in gang notoriety
 - (string) name of the notoriety team that was affected, or "None" if all scores were affected
