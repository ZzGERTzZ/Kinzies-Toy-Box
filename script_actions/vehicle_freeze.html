vehicle_freeze(vehicle_name[, zero_velocity])
Parameters:
vehicle_name (string) name of the vehicle to stop
zero_velocity (boolean, optional) TRUE if the linear and angular velocity should be set to 0, FALSE otherwise (defaults to FALSE)
 
Returns:
none
Description:
Forcefully stop a vehicle at its current location
