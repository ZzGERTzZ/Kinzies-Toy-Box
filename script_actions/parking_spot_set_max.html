parking_spot_set_max(new_max)
Parameters:
new_max (integer) Number of parked cars that will spawn.
 
Returns:
none
Description:
Sets the maximum number of parked cars that will spawn. The default value is MAX_PARKING_SPAWN_VEHICLES (18 at the time this was written).
