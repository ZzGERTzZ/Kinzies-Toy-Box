mission_set_checkpoint(checkpoint_name[, ignore_vehicles[, ignore_notoriety]])
Parameters:
checkpoint_name (string) name of the checkpoint
ignore_vehicles (boolean, optional) set to true to not restore the player's vehicle at the checkpoint (defaults to false)
ignore_notoriety (boolean, optional) set to true to not restore the player's notoriety at the checkpoint (defaults to false)
 
Returns:
none
Description:
Set the most recent mission checkpoint that the player has passed.
