trigger_get_radius(trigger_name)
Parameters:
trigger_name (string) name of the script trigger
 
Returns:
(float) radius (or half extents) of the trigger
Description:
Get the radius of a script trigger (return radius of cylinder and sphere trigger types, and half extents for box types)
