mesh_mover_set_damaged_by_player_only(name, invulnerable)
Parameters:
name (string) name of the mesh mover
invulnerable (boolean) True to set the mesh mover only damaged by player, false otherwise
 
Returns:
(boolean) true if the mover was found and the flag was set, false otherwise.
Description:
Sets the flag on the mover for if this can only be damaged by the player
