get_vehicle_type(handle)
Parameters:
handle (integer) handle of the vehicle
 
Returns:
none
Description:
Get the vehicle type of a vehicle.

The possible vehicle types are:

 VT_AUTOMOBILE
 VT_MOTORCYCLE
 VT_AIRPLANE
 VT_HELICOPTER
 VT_WATERCRAFT
 VT_NONE (vehicle does not exist)
