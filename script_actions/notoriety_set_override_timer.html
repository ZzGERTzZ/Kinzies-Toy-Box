notoriety_set_override_timer(group_name, disable, min_time, max_time)
Parameters:
group_name (string) name of the group
disable (boolean) set to true to override time otherwise false
min_time (int) min time in milliseconds to set new spawn interval
max_time (int) max time in milliseconds to set new spawn interval
 
Returns:
none
Description:
Set the override timer of a notoriety spawn group.
