hud_set_fake_notoriety(name, enabled[, level])
Parameters:
name (string) notoriety team
enabled (boolean) set to true to enable fake notoriety, or false to disable
level (float, optional) fake notoriety level to set to (defaults to 0.0)
 
Returns:
none
Description:
Set the current fake notoriety level.
