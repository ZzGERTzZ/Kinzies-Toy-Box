set_dont_attack_me_on_sight_flag(name[, dont_attack])
Parameters:
name (string) name of the NPC
dont_attack (boolean, optional) set to true to disallow the NPC to be attacked on sight by other NPCs, or false to allow (defaults to true)
 
Returns:
none
Description:
Set whether an NPC may be attacked on sight by other NPCs.
