ai_do_scripted_rush(attacker[, target])
Parameters:
attacker (string) name of the attacker
target (string, optional) name of the target, navpoint, (null => current target)
 
Returns:
none
Description:
Make a character do a combat rush - run to destination, only looking at targets, not firing.
