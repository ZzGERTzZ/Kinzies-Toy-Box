set_perfect_aim(npc_name[, perfect_aim])
Parameters:
npc_name (string) name of the NPC
perfect_aim (boolean, optional) set to true to give the NPC perfect aim, or false to use normal aim (defaults to true)
 
Returns:
none
Description:
Enable/disable perfect aim for an NPC.

An NPC with perfect aim enabled will always hit the target when firing a gun, and will always lead the target perfectly when firing a rocket launcher (assuming the target's velocity remains constant).
