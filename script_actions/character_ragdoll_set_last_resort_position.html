character_ragdoll_set_last_resort_position(nav_name)
Parameters:
nav_name (string) Name of the nav point to use as the last resort teleport location.
 
Returns:
none
Description:
Sets a position to be used as a last resort teleport destination if all better methods of recovering from a ragdoll are exhausted during a mission. This position is used instead of using a warp to shore nav point, and is useful in SR3 for last-ditch recoveries while on the Daedalus or on Mars.
