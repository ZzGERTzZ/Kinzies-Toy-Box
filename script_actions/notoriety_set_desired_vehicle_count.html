notoriety_set_desired_vehicle_count(team_name[, vehicle_count])
Parameters:
team_name (string) name of the team
vehicle_count (integer, optional) number of notoriety vehicles, or -1 to reset to normal (defaults to -1)
 
Returns:
none
Description:
Set/reset the number of notoriety vehicles that can spawn at any given time.
