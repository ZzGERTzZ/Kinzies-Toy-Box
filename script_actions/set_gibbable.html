set_gibbable(name[, gibbable])
Parameters:
name (string) name of the NPC
gibbable (boolean, optional) set to true if the NPC should be gibbable, or false otherwise (defaults to true)
 
Returns:
none
Description:
Set whether an NPC can be instagibbed by the sonic gun.
