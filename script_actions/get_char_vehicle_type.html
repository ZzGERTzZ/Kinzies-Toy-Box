get_char_vehicle_type(name)
Parameters:
name (string) name of the character
 
Returns:
(enumeration) type of the vehicle
Description:
Get the type of the vehicle being driven by a character.

The possible vehicle types are:

VT_AUTOMOBILE VT_MOTORCYCLE VT_AIRPLANE VT_HELICOPTER VT_VTOL VT_WATERCRAFT VT_NONE (character is not in a vehicle)
