hud_bar_on(index, skin_name[, label_name[, max_value[, sync_flags]]])
Parameters:
index (integer) index of the bar to show
skin_name (string) name of the skin to use (see below)
label_name (string, optional) override label to display instead of the skin's label (defaults to "")
max_value (float, optional) the maximum value that the bar displays (defaults to 0)
sync_flags (integer, optional) synchronization type (MULTI_SYNC_LOCAL = affects local player, MULTI_SYNC_REMOTE = affects remote player, or MULTI_SYNC_ALL = affects both players; defaults to MULTI_SYNC_ALL)
 
Returns:
none
Description:
Show a GSI bar.

The available colors are found in vdo_gsi.lua. For convenience, I've copied the list here. Notice that "Damage" is yellow, just like "Default". If you want red, use "Health".

 Default            --Default: Yellow
 Health            --Description: Health
 Damage            --Description: Yellow
 Radioactivity      --Description: NEON Greeen
 Media            --Description: Blue
 Taunt            --Description: Yellow
 Fear               --Description: Yellow
 Pleasure         --Description: Pink (Escort)
 Footage            --Description: Blue (Escort)
 Mayhem            --Description: Mayhem has special properties where it grabs the colors from hud_mayhem.lua
 TankMayhem         --Description: Mayhem has special properties where it grabs the colors from hud_mayhem.lua
 Fight_Club         --Description: Fight club is basically a red bar but requires special update functionality because the label changes all the time.
 Nitrous            --Description: Blue
