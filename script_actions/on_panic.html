on_panic(func_name[, name])
Parameters:
func_name (string) name of the function, or "" to unregister the event
name (string, optional) name of the NPC
 
Returns:
none
Description:
Register a callback function for when an NPC panics.

When the callback is executed, the following parameters are passed to it:
 - (string) name of the NPC
 - (float) duration of the panic, in seconds

NPCs will enter panic AI mode if set on fire, hit with pepper spray or a flashbang, of stuck with a satchel charge.
