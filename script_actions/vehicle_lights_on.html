vehicle_lights_on(vehicle_name[, force])
Parameters:
vehicle_name (string) name of the vehicle
force (boolean, optional) set to true to enable forcing of the vehicle lights on, or false to disable forcing of the vehicle lights on (defaults to true)
 
Returns:
none
Description:
Set whether a vehicle's lights should be forced on.
