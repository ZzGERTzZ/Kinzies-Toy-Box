ai_do_scripted_take_cover(attacker[, target])
Parameters:
attacker (string) name of the attacker
target (string, optional)
 
Returns:
none
Description:
Make a character go to cover. If you want them to stay there I suggest using with ai_set_in_scripted_cover(name,false) and npc_set_leash_to_nav(name, nav, dist)

NULL = AI finds node near npc
                           nav_point = AI finds node near and same direction of navpoint.
                           cover_node = goes to cover node

NOTE: use npc_set_leash_to_nav() to make them stay in that area. And ai_set_in_scripted_cover() to heavily use cover.
