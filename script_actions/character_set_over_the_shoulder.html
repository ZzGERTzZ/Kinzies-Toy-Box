character_set_over_the_shoulder(name[, new_state])
Parameters:
name (string) name of the character
new_state (boolean, optional) set to true to make the human enter human shield mode when grabbed, false to be used as a normal human sheild
 
Returns:
none
Description:
Enable/disable the use of the human as a over the shoulder human shield.
