character_set_never_catch_fire(name[, no_panic])
Parameters:
name (string) name of the character
no_panic (boolean, optional) set to true to force a character to never catch on fire (defaults to TRUE)
 
Returns:
none
Description:
Enable/disable whether an NPC can catch on fire (never_ignite)
