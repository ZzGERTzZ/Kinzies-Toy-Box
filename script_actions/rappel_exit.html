rappel_exit(human_name[, force_warp[, navpoint_name]])
Parameters:
human_name (string) name of the human
force_warp (bool, optional) if the human should be warped to the ground, even if a navpoint_name isn't specified.
navpoint_name (string, optional) name of the navpoint the human should be teleported to when exiting this state
 
Returns:
none
Description:
Set a human to exit the rappelling state
