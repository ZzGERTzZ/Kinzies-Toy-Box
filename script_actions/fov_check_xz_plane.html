fov_check_xz_plane(observer, observed)
Parameters:
observer (string) name of the observing NPC
observed (string) name of the target character
 
Returns:
(boolean) true if the target is within the view, else false
Description:
Check if a character is within the observation FOV of an NPC, ignoring the vertical (y) axis.
