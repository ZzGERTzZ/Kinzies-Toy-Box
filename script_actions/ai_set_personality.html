ai_set_personality(npc_name, personality_name)
Parameters:
npc_name (string) name of the NPC
personality_name (string) data/tables/AI/ai_personlity.xtbl entry. Please request changes to the table thru the AI team.
 
Returns:
none
Description:
Change the personality of an NPC

invalid string will reset it default.
                  Most common uses: "gang defensive", "gang offensive", "police defensive", "police offensive"
