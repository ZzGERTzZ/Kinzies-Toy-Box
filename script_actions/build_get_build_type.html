build_get_build_type()
Parameters:
none
 
Returns:
(integer) enumeration corresponding to the build type (1 = normal, 2 = Australia, 3 = German, 4 = Japan 360, 5 = Japan PS3)
Description:
Get the game localization build type.

This function is used for localizing scripted gameplay.
