character_disable_revival(name[, dont_revive])
Parameters:
name (string) name of the character
dont_revive (boolean, optional) set to true to disable the ability to go into the downed state, or false to enable it (defaults to true)
 
Returns:
none
Description:
Enable/disable the ability for a character to be revived
