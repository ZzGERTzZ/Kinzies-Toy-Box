ai_suggest_action(attacker, action_name, use_object_name)
Parameters:
attacker (string) name of the attacker
action_name (string) name of the action to suggest, see tables/AI/combat_actions.xtbl for list.
use_object_name (opt string) name something to use based on the contexted of the action IE: nav point, human, vehicle, path, etc.
 
Returns:
none
Description:
Make a character do a action next time they can, IE: take cover ASAP, I don't care where.
